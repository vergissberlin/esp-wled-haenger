# WLED for the ESP8266 with infrared remote control

This repository contains presets for the WLED project for the ESP8266 with infrared remote control.

## Remotes

### Apple iMac G5

Menu:     0x77E14050
Play:       0x77E12050
Up:         0x77E1D050
Down:    0x77E1B050
Left:       0x77E11050
Right:    0x77E1E050

### Apple TV 2

Menu:    0x77E1C02F
Play:      0x77E1FA2F
Middle:  0x77E13A2F
Up:         0x77E1502F
Down:    0x77E1302F
Left:       0x77E1902F
Right:     0x77E1602F
